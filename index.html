Styling with CSS

How to make your webpage more appealing!

The CSS properties specifies what to style of the targeted HTML elements

CSS Properties Definition Example
Color This is the color of the text This is blue
Background color This is the color behind the text This background is yellow
Font-size This is the size of your font 20px 40px
Font-weight This is how "heavy" your font will appear This is bold
There are many more!
